Smart Lender – Loan Eligibility Prediction System

🌐 Live Demo

https://smartfund-1.onrender.com/

📖 Project Overview

Smart Lender is a Machine Learning-based web application that predicts whether an applicant is eligible for a loan. The application analyzes applicant details using trained machine learning models and provides quick and accurate loan eligibility predictions through a user-friendly web interface.

🚀 Technologies Used

- Python
- Flask
- Scikit-learn
- XGBoost
- Pandas
- NumPy
- HTML5
- CSS3
- Bootstrap
- Joblib

✨ Features

- Loan Eligibility Prediction
- Responsive and User-Friendly Interface
- Data Preprocessing
- Multiple Machine Learning Models
- Fast and Accurate Predictions
- Cloud Deployment

🤖 Machine Learning Models

- Decision Tree Classifier
- Random Forest Classifier
- K-Nearest Neighbors (KNN)
- XGBoost Classifier

🏆 Best Performing Model

XGBoost achieved the highest prediction accuracy and was selected as the final model for deployment.
